Cesium demo

This demo showcases a variety of functions:

  • The default map view shows a 3D model tiled and loaded via Cesium Ion
  • Clicking twice on the map will drop start/end pins, and then displays the shortest route, calculated via ESRI's routing API
  • Clicking the button below will toggle on/off the Google 3D Buildings tileset, loaded via Cesium Ion
